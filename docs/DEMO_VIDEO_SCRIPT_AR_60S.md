# ChefHub — سيناريو فيديو عرض سريع (60 ثانية) — عربي

> الهدف: فيديو مختصر جدًا يوضح القيمة خلال دقيقة.

## اللقطات + النص الصوتي (Voiceover)

### 0:00–0:05 — افتتاحية
**على الشاشة:** الصفحة الرئيسية `/`
**نص على الشاشة:** "ChefHub — منصة الشيف المنزلي"
**الصوت:**
"ChefHub منصة تربط العملاء بشيفات منزليين… بتجربة طلب سهلة وإدارة كاملة." 

### 0:05–0:20 — العميل (تصفح + سلة)
**على الشاشة:** `/dishes` → فتح طبق → إضافة للسلة → `/checkout`
**نص على الشاشة:** "للعميل: تصفح → اختيار → Checkout"
**الصوت:**
"العميل يتصفح الأصناف المتاحة فقط… يفتح تفاصيل الطبق… ويضيفه للسلة ويوصل للـ Checkout بسرعة." 

### 0:20–0:32 — صفحة الشيف
**على الشاشة:** `/chef` → افتح `/chef/[id]`
**نص على الشاشة:** "ملف الشيف + الأصناف + العروض"
**الصوت:**
"ومن ملف الشيف… تشوف معلوماته وتقييمه… وأطباقه المتاحة… والعروض الخاصة الموسمية." 

### 0:32–0:50 — لوحة الشيف
**على الشاشة:** (حساب شيف) `/chef/dishes` ثم لمحة سريعة `/chef/dishes/new`
**نص على الشاشة:** "للشيف: إضافة صنف + صور + تفعيل"
**الصوت:**
"الشيف يدير أصنافه بسهولة… يضيف صنف مع صور… ويتحكم بالتفعيل والتوفر." 

### 0:50–0:58 — الأدمن (الجودة)
**على الشاشة:** `/admin/chef` ثم لمحة `/admin/dashboard`
**نص على الشاشة:** "للأدمن: اعتماد الشيف + إحصائيات"
**الصوت:**
"والأدمن يضمن الجودة… يعتمد الشيفات ويتابع الإحصائيات." 

### 0:58–1:00 — خاتمة
**على الشاشة:** رجوع للصفحة الرئيسية
**نص على الشاشة:** "جاهز للعرض والتوسع"
**الصوت:**
"ChefHub… جاهز للعرض والتوسع."
