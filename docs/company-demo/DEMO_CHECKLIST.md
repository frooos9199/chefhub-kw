# Demo Checklist (قبل عرض الشركة)

## 1) تشغيل وتجهيز
- [ ] `npm install` تم بنجاح
- [ ] `.env.local` موجود ويحتوي مفاتيح Firebase المطلوبة
- [ ] `npm run build` ناجح
- [ ] حسابات الديمو جاهزة (عميل/شيف/أدمن)

## 2) بيانات الديمو
- [ ] يوجد شيف واحد على الأقل حالته `approved` (أو `active` للبيانات القديمة)
- [ ] يوجد أطباق للشيف: `isActive=true` و `isAvailable=true`
- [ ] يوجد صورة افتراضية أو صور حقيقية للأطباق

## 3) الصفحات الأساسية
- [ ] الرئيسية تعرض أقسام الشيفات والأطباق
- [ ] صفحة الشيفات `/chef` تعرض قائمة بدون أخطاء
- [ ] صفحة التصفح `/browse` تعرض الشيفات/الأطباق مع فلترة
- [ ] صفحة تفاصيل الشيف `/chef/[id]` تعمل
- [ ] صفحة الأطباق `/dishes` تعمل

## 4) تدفق الطلب
- [ ] إضافة للسلة تعمل
- [ ] Checkout يعمل
- [ ] صفحة نجاح الطلب تعمل

## 5) إدارة الشيف
- [ ] لوحة الشيف تعمل
- [ ] إضافة/تعديل طبق يعمل

## 6) إدارة الأدمن
- [ ] شاشة إدارة الشيفات تعمل
- [ ] اعتماد شيف جديد يعمل

## 7) الإشعارات (اختياري حسب العرض)
- [ ] In-app notifications تظهر
- [ ] Email يعمل عند توفر `SENDGRID_API_KEY`
- [ ] WhatsApp يعمل عند توفر `TWILIO_*`

## 8) التواصل
- [ ] رابط المطور: https://www.q8nexdev.com
