# Demo Storyboard — ChefHub (90s)

## Shots (لقطات) جاهزة للتسجيل
1. Home: `/` (Hero + قسم أطباق/عروض)
2. Dishes listing: `/dishes` (فلترة/بحث سريع) ثم فتح طبق
3. Checkout: `/checkout` (السلة) 
4. Chefs listing: `/chef` ثم فتح ` /chef/[id]`
5. Chef dishes dashboard: `/chef/dishes`
6. Add dish: `/chef/dishes/new` (اختيار صور + حفظ)
7. Chef special orders: `/chef/special-orders` ثم `/chef/special-orders/new`
8. Admin dashboard: `/admin/dashboard`
9. Admin chefs: `/admin/chef` ثم approve/suspend من صفحة شيف

## On-screen overlays (نصوص قصيرة)
- منصة الشيف المنزلي
- للعميل: تصفح → اختيار → سلة
- للعميل: ملف الشيف + الأصناف
- للشيف: إدارة الأصناف + الصور
- للشيف: عروض موسمية
- للأدمن: اعتماد + رقابة

## Tips
- كل لقطة 5–12 ثانية.
- انتقل بين الصفحات بسرعة (بدون انتظار طويل).
- لو في بيانات قليلة: جهز "حسابات Demo" مسبقًا أو اعرض Mock sections.
